You can contribute your experience by sharing tips, strategy, guide,... You just have to follow these simple steps to get your post appear on the site!

Step 1: Go to this link and create an issue

Step 2: Set the label

Step 3: Write your content

Step 4: Submit!

Arena of Evolution Strategy Guide is a Jekyll website hosted on Github. Here is the repository link: evolutionchess.github.io

You don't have to submit an issue

Issues is not a place for sharing content, we sorry for it! But sadly not everyone can use Jekyll and Git so we borrow Issues as a place to post contribution. Feel free to share your content via Issue, or create a new post directly in the repository's source code (it can be a HTML or a Markdown file inside _posts folder) and make a pull request. We will take a review and your post will be published as soon as we saw it.

If you need any special permission, tell us your Github account and we will grant it for you!

Feel free to contribute anything, from content to source code!

Our website have weird looking, it's true. Feel free to be our designer, write some css and make a pull request. We love having your contribution!

Our stats is outdated, sorry about that! Feel free to update the stats, they are stored as csv files inside _data folder. You can send us a pull request after you finish the update or tell us to grant you write permisson and you can push the code directly into the master branch.
